Responsive Web Design (RWD) is a web design approach that makes a website automatically adjust its layout, content, images, and spacing according to different screen sizes and devices.

This will make website responsive
1.display:flex;
2.display:grid;
3.@media()



Media Queries are CSS rules used to apply different styles based on the screen size, device type, or viewport width.

Syntax
@media(condition){
CSS Code
}

max-width
@media(max-width:768px)
min-width
@media(min-width:768px)
