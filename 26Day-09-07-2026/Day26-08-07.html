Child → Parent Communication--- Child to Parent Communication is the process of sending data from a Child Component to its Parent Component using a callback function passed through Props.

Child → Child Communication---- Child to Child Communication is the process of sharing data between sibling components through their common Parent Component.

Callback Function in React---- A callback function is a function passed to another component or function so that it can be called later when a specific event occurs.

Local State is state that is created, managed, and used within a specific component. It is commonly managed using the useState Hook in functional components and causes the component to re-render when updated.

Local State is component-specific data managed internally by a component, usually using the useState Hook.

Global State :
Global State is data that needs to be shared and accessed by multiple components in an application.

Props Drilling is the process of passing data through multiple intermediate components just to reach a deeply nested component.
